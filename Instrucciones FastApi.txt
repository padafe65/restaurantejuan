2. Instalación de la "Maquinaria" (Librerías)
Instalamos las piezas del motor una por una según nos las pidió el sistema:

pip install fastapi uvicorn: El corazón del Backend y el servidor que lo corre.

pip install sqlalchemy pymysql cryptography: El puente para que Python hable con tu base de datos MySQL y los datos no se borren al apagar la PC.

pip install "python-jose[cryptography]" passlib[bcrypt]: Las llaves de seguridad para encriptar contraseñas y manejar los roles de Admin y Mesero.

pip install python-multipart: Permite que el sistema reciba datos desde los formularios de inicio de sesión (Login).

3. Ejecutar Streamlit
Como ya instalaste todas las librerías necesarias anteriormente (incluyendo streamlit, pandas y requests), usa el siguiente comando para lanzar la interfaz de usuario:

PowerShell
python -m streamlit run frontend/app_web.py
¿Por qué hacerlo así?
Uso de python -m: Al igual que con el Backend, esto garantiza que se use la versión 3.11 de tu entorno virtual y no la 3.14 del sistema, evitando errores de "módulo no encontrado".

Comunicación Interna: Tu archivo gestion_reservas.py utiliza la librería requests para conectarse a http://127.0.0.1:8000 (tu Backend). Si el Backend no está encendido, el Frontend mostrará errores de conexión.
